# 数字人微课 - 测验与练习功能更新

## 更新内容

为数字人微课页面实现了完整的测验与练习功能，围绕"电力是什么？电从哪里来？"主题。

## 新增功能

### 1. FlashCard 记忆卡片（5张）

**主题：电力知识**

卡片内容：
1. **什么是电力？**
   - 答案：电力是一种能量形式，由电荷的移动产生。它可以转化为光、热、动力等多种形式，是现代生活中不可或缺的能源。

2. **电从哪里来？**
   - 答案：电主要来自发电厂。发电厂通过燃烧煤炭、利用水力、风力、太阳能或核能等方式，将其他形式的能量转化为电能。

3. **电是如何传输到我们家里的？**
   - 答案：电通过高压输电线从发电厂传输到变电站，然后通过配电网降低电压，最后通过电线传输到每家每户。

4. **为什么要节约用电？**
   - 答案：节约用电可以减少能源消耗，保护环境，降低碳排放。同时也能节省电费，培养良好的生活习惯。

5. **生活中哪些电器最耗电？**
   - 答案：空调、电热水器、电暖器等大功率电器最耗电。合理使用这些电器，可以有效节约用电。

**功能特点：**
- 点击卡片可翻转查看答案
- 3D翻转动画效果
- 支持上一张/下一张导航
- 显示当前卡片进度

### 2. Quiz 快速测验（5道题）

**主题：电力知识测验**

题目内容：
1. **电力是一种什么形式的能量？**
   - 选项：机械能、化学能、电能、热能
   - 正确答案：电能
   - 解析：电力是电能的一种表现形式，由电荷的移动产生。

2. **以下哪种方式不能用来发电？**
   - 选项：水力发电、风力发电、声音发电、太阳能发电
   - 正确答案：声音发电
   - 解析：声音的能量太小，无法有效地转化为电能。常见的发电方式包括水力、风力、太阳能、火力和核能发电。

3. **家里的电是通过什么传输过来的？**
   - 选项：水管、电线、空气、光纤
   - 正确答案：电线
   - 解析：电通过电线传输。电从发电厂经过高压输电线、变电站，最后通过配电网的电线传输到每家每户。

4. **以下哪个行为可以节约用电？**
   - 选项：长时间开着电视、离开房间关灯、空调温度调到最低、电脑一直不关机
   - 正确答案：离开房间关灯
   - 解析：离开房间时关灯是节约用电的好习惯。其他选项都会造成不必要的电力浪费。

5. **为什么触摸电源插座很危险？**
   - 选项：会被电到、插座会坏、会着火、没有危险
   - 正确答案：会被电到
   - 解析：触摸电源插座非常危险，可能会触电。电流通过人体会造成伤害，严重时甚至危及生命。所以千万不要用手触摸电源插座！

**功能特点：**
- 单选题形式
- 实时显示得分
- 答题后显示正确答案和解析
- 答对/答错有不同的视觉反馈
- 完成后显示总分和评价
- 支持重新测验

## 技术实现

### 状态管理
- 使用 React Hooks 管理闪卡和测验状态
- 独立的状态控制确保两个功能互不干扰

### UI/UX 设计
- 模态框形式展示，不影响主页面
- 响应式设计，适配不同屏幕尺寸
- 流畅的动画效果提升用户体验
- 清晰的视觉反馈（颜色编码）

### 交互设计
- **闪卡**：点击翻转，左右导航
- **测验**：选择答案 → 提交 → 查看解析 → 下一题
- 完成后可重新开始

## 使用方法

1. 访问数字人微课页面
2. 滚动到底部"测验与练习"区域
3. 点击"FlashCard 记忆卡片"或"Quiz 快速测验"
4. 在弹出的模态框中进行学习或测验
5. 完成后可关闭或重新开始
