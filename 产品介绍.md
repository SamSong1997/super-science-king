# 超级理科王 - 产品介绍

## 产品定位

**超级理科王**是一个面向中小学生的在线科学学习平台，通过电子书、视频微课、互动实验、游戏化学习等多种形式，全方位提升学生的科学素养和学习兴趣。

---

## 核心功能

### 1. 📚 电子书阅读器
- **在线阅读**：支持PDF格式，流畅的翻页体验
- **智能导航**：知识点快速跳转，章节目录一键定位
- **学习工具**：高亮标注、笔记记录、书签收藏
- **进度追踪**：自动保存阅读进度，随时继续学习
- **护眼模式**：夜间模式和护眼配色，保护视力

### 2. 🎥 数字人微课
- **名师讲解**：真人出镜，生动有趣的教学方式
- **碎片学习**：每节5-10分钟，适合碎片化时间
- **灵活控制**：倍速播放、进度拖拽、重点回看
- **知识关联**：与电子书知识点无缝衔接
- **多学科覆盖**：物理、化学、生物、数学全覆盖

### 3. 🔬 互动实验室
- **虚拟实验**：基于PhET平台的3D可视化实验
- **安全探索**：无需实验器材，零风险操作
- **实时反馈**：即时观察实验现象和数据变化
- **实验报告**：自动生成实验记录和分析
- **多学科实验**：物理、化学、生物实验全覆盖

### 4. 📝 科学素养测评
- **9维度评估**：观察、分类、推理、实验设计、数据分析、模型建构、解释、论证、质疑创新
- **个性化报告**：精准定位薄弱环节，提供针对性建议
- **进步追踪**：定期测评，可视化展示成长轨迹
- **智能推荐**：根据测评结果推荐学习内容

### 5. 🎮 知识闯关游戏
- **火柴棒谜题**：移动火柴使等式成立，锻炼逻辑思维
- **找不同**：观察力和专注力训练
- **华容道**：空间想象和策略规划能力
- **积分排行**：与同学PK，激发学习动力
- **知识融合**：每个关卡融入科学知识点

### 6. 💬 AI智能助教
- **24小时在线**：随时随地解答疑问
- **多种交互**：文字、语音、图片识别
- **个性化建议**：根据学习数据提供定制化学习计划
- **知识库丰富**：覆盖中小学全学科知识点

## 技术架构

### 前端技术
- **框架**：Next.js 16 (App Router) + React 19
- **样式**：Tailwind CSS 4 + Radix UI
- **PDF阅读**：@react-pdf-viewer
- **图表**：Recharts
- **动画**：Framer Motion

### 后端技术
- **API**：Next.js API Routes
- **数据库**：SQLite + Prisma ORM
- **文件存储**：本地存储 / 云存储（可选）

### 部署方案
- **推荐平台**：Vercel / 自建服务器
- **数据库**：SQLite（开发）/ PostgreSQL（生产）
- **CDN**：静态资源加速

## 用户场景

### 学生端
1. **课前预习**：通过电子书和微课了解新知识
2. **课后复习**：做题测评，巩固所学内容
3. **实验探索**：在虚拟实验室中动手操作
4. **游戏闯关**：在娱乐中学习，提升兴趣
5. **AI答疑**：遇到问题随时提问

### 教师端
1. **内容管理**：上传电子书、视频、题目
2. **学情分析**：查看学生学习数据和报告
3. **作业布置**：推送学习任务和测评
4. **进度监控**：实时了解学生学习进度

### 管理员端
1. **用户管理**：管理学生和教师账号
2. **内容审核**：审核上传的资源
3. **数据统计**：查看平台使用情况
4. **系统配置**：调整平台参数和设置

## 数据模型

### 核心数据表
- **用户表**：学生、教师、管理员信息
- **电子书表**：书籍信息、PDF路径、封面
- **知识点表**：关联电子书，包含页码
- **章节表**：关联电子书，包含页码
- **阅读进度表**：记录用户阅读位置
- **题目表**：题目、选项、答案、解析
- **测评结果表**：用户答题记录和得分
- **实验表**：实验信息、PhET链接、操作步骤
- **游戏关卡表**：关卡数据、答案、提示

## 部署流程

### 开发环境
```bash
# 安装依赖
npm install

# 初始化数据库
npm run db:push
npm run db:seed

# 启动开发服务器
npm run dev
```

### 生产部署
```bash
# 构建生产版本
npm run build

# 启动生产服务器
npm start
```

### Vercel一键部署
点击按钮即可部署到Vercel平台，自动配置域名和HTTPS。
